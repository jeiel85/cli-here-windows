# Branding

## Names

| Item | Value |
|---|---|
| English name | CLI Here |
| Korean name | CLI 여기서 열기 |
| Repository | cli-here |
| Executable | CliHere.exe |
| Release ZIP | CliHere-win-x64.zip |

## Tagline

English:

> Launch your AI coding CLI from File Explorer.

Korean:

> Windows 탐색기에서 AI CLI를 바로 실행하세요.

## Positioning

CLI Here is a lightweight utility for Windows developers who use terminal-based AI coding CLIs.

It should feel:

- practical
- lightweight
- trustworthy
- developer-focused
- not over-engineered

## Distribution

Use English-first messaging for GitHub and Reddit. Provide Korean documentation through `README.ko.md`.

## Naming rule

Use `CLI Here` in English-facing contexts and `CLI 여기서 열기` in Korean-facing contexts. When introducing the product globally, use:

```text
CLI Here / CLI 여기서 열기
```
