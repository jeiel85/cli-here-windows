# Roadmap

## v0.1.0 - Initial MVP

- WPF settings UI
- Korean and English localization
- System/Korean/English language selection
- CLI detection from PATH
- Gemini, OpenCode, Claude Code, and Codex support
- Context menu add/remove
- Folder background right-click support
- Folder right-click support
- PowerShell launch
- Windows Terminal launch
- Optional administrator launch
- JSON settings
- GitHub Actions build/test
- ZIP release artifact

## v0.2.0

- Parent context menu group
- Custom CLI definitions
- PowerShell 7 support
- Better icons
- Detailed CLI diagnostics
- Repair context menu action

## v0.3.0

- Portable mode
- More terminal profiles
- Better accessibility

## v0.4.0

- CLI 프로파일 저장 기능 (자주 사용하는 CLI 설정 저장)
- CLI 실행 이력 관리
- 테마 커스터마이징 (다크/라이트 모드)
- CLI 그룹 기능 (폴더별 CLI 그룹핑)
- CLI 실행 로그
- CLI 버전 표시
