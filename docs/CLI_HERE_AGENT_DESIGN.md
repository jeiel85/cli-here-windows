# CLI Here / CLI 여기서 열기 — Integrated Design Summary

## Repository

Recommended repository name:

```text
cli-here
```

## Product

`CLI Here / CLI 여기서 열기` is a lightweight Windows utility for launching AI coding CLIs from File Explorer's right-click menu.

## Core CLIs

- Gemini CLI
- OpenCode
- Claude Code
- OpenAI Codex CLI

## Core stack

- C#
- WPF
- .NET 10 LTS
- xUnit
- GitHub Actions

## Main principles

- English-first public README
- Korean documentation in README.ko.md
- Korean and English UI from day one
- HKCU-only registry integration
- No telemetry
- No automatic third-party CLI installation
- Optional launch-time administrator mode
- GitHub Actions-centered validation
- Release asset verification for `CliHere-win-x64.zip`
